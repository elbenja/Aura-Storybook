# Aura Design System

Welcome to the **Aura** component library — the design system for the Aura real estate and property analysis platform.

## Foundation

This library is built on:

- **shadcn/ui** — Accessible, composable base components
- **Tailwind CSS v4** — Utility-first styling with design tokens
- **HugeIcons** — 4,600+ free icons
- **React 19** + TypeScript

## Getting Started

Browse the sections in the sidebar to explore the design system. Each component includes:

- Interactive controls to adjust props
- Auto-generated documentation
- Accessibility checks (a11y)

Use the **theme toggle** in the toolbar to switch between light and dark mode. Use the **viewport** selector for mobile/desktop breakpoints.

## Structure

| Category | Description |
|----------|-------------|
| **Foundations** | Typography, Colors, and Icons — the design tokens |
| **Primitives** | Base shadcn/ui components (Button, Card, Badge, etc.) |
| **Composites** | Combined components built from primitives |
